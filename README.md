# Mein interaktives Jupyter Book

Gebaut mit [Jupyter Book v2 / MyST](https://jupyterbook.org/stable) und interaktiver
Ausführung via [Binder](https://mybinder.org).

## Lokale Entwicklung

```bash
npm install -g mystmd
myst start
```

## Deployment

Bei jedem Push auf `main` wird die Website automatisch via GitHub Actions auf
GitHub Pages deployed.
